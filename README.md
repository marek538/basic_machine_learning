# Titanic Survival Prediction (Binary Classification)

## Project Description
A binary classification project to predict the survival of passengers on the Titanic. The core focus is on **data preprocessing, feature engineering, and evaluating predictive models** to understand what factors most influenced survival rates.

## Models used
* Decision Tree
* K-Nearest Neighbors

# Life Expectancy Prediction (Regression Analysis)

## Project Description
A regression analysis project focused on predicting life expectancy across various countries and years. The model leverages socio-economic and health indicators (such as Adult Mortality, Alcohol consumption, and healthcare expenditure) to forecast health outcomes.

## Models used
* Linear Regression
* Random Forest Regressor
